Git commands
Command Description
get init Initialize a local Git repository
git status Check status
git add <file-name> Add a file to a staging area
git commit -m "<commit message> Commit changes
git rm -r <file/folder> Remove a file or a folder
git push Push changes to remote repository
git log Check all commits registered
git branch Says what is the actual branch
git branch <name> Create a new branch
git branch -a Copy from remotes branches
git checkout <branch> Change the branch
git remote set-url origin <path> Change the remote repository
