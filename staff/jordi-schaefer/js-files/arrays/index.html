Ejercicios arrays

Funciones testing

at

recive un indice y devuelve el item que se encuentra en el index deseado

concat

devuelve la union de 2 o mas arrays

copyWithin

copia partes del array dentro lugar del array, sin modificar longitud (X, Y ,Z) x=donde copio, y=inicio tramo, z=final tramo

every

devuelve true si todos los elementoscumplken la funcion callback

fill

fill(value, start, end) / cambia los elementos del array desde y/o hasta una posicion definida

filter

crea un nuevo array con todos los elementos que cumplan la condicion deseada

find

devuelve el valor del primer elemento que cumple la funcion

flat

crea un nuevo array uniendo los sub-elementos de cada objeto (si se especifica un numero es el nivel de sub-elemento que concatena)

forEach

modifica un array implementando en cada valor la funcion deseada

groupBy

agrupa

includes

determina si un array incluye un valor especifico o no, devolviendo true o false

indexOf

devuelve el primer indice en el que se encuentra el valor deseado, o en su defecto devuelve -1

join

devuelve un string que contiene los elementos del array separado por el elemento deseado, o en su defecto comas

lastindexOf

devuelve el ultimo indice en el que se encuentra el elemento que quiere ser encontrado

map

crea un nuevo array implementando en cada valor la funcion deseada

pop

elimina el ultimo elemento del array y lo devuelve

push

añade elementos al final del array

reduce

ejecuta una funcion callback para cada valor del array y su total anterior, devuelve el total de la funcion

reverse

devuelve un array nuevo con las posiciones inversidas

shift

elimina el primer elemento del array y lo devuelve, acortando la longitud del array original

slice

slice(inicio, fin) devuelve un nuevo array que es el recorte del original desde y hasta las posiciones dadas

some

devuelve true si hay algun valor que cumpla la funcion callback

sort

orderna un array

splice

slice(X, Y, Z): a partir de la posicion X (incluida) remplaza Y elementos por el Z elemento

unshift

añade 1 o mas elementos al principio del array, y devuelve el nuevo largo del array
