Basch Commands

help mostrar los comandos
cd <path> comando para moverse entre carpetas
cd .. volver atras en la carpetsa
mkdir crear carpeta
  • eliminar directorio: rd
  • eliminar archivo: rmdir
  • mostrar subcarpetas: dir <path>
  • limpiar ventana: clear
  • copiar uno o mas archivos: copy archivo destino
  • mover archivos: move archivo destino
  • cambiar nombre archivo: rename archivo
  • editor simple de archivo: nano <archivo>
  • abrir archivo html: start <archivo>
  • leer archivo: cat <archivo>
  • abrir editor de codigo de la carpeta actual: code .
  • eliminar archivo r=remove f=force: rm -rf archivo
  • ver archivos ocultos: ls -a
