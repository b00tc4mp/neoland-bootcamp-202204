git config --global user.email''

Configura el correo del usuario de manera global

git config -h

Devuelve todas las opciones y posibles configuraciones

git branch

Crea una nueva rama

git checkout

Cambia a una rama existente

git branch -D 'nombreDeRama'

Borra la rama que escribo en las comillas

git branch -a

Muestra las ramas en seguimiento remoto y las locales

git --version

The command to check which version of Git you’re using is the same on both Windows and Mac.

git commit -m ''

Set configuration values for your username, email, gpg key, preferred diff algorithm, file formats and more.

git status

The git status command displays the status of the working directory and staging area. Lets you see which changes have been staged, which ones haven't, and which files Git isn't going to track.

git push

The git push command is used to push content from the local repository to a remote repository.

git add

selects that file and moves it to the staging area, marking it for inclusion in the next commit.

git add .

selects that file and moves it to the staging area, marking it for inclusion in the next commit.

git origin

it is simply the default name given to the main remote repository we are working against.

git rm

The git rm command can be used to remove single files or a set of files.

git restored

Restore specified paths in the working tree with some contents from a restore source

git mv

Move or rename a file, directory or symlink.

git clone 'url'

You clone a repository
